# Groww Pulse Content Standards

1. **Theme Phrasing**: 
   - Never use one-word crude themes (e.g., "Useless", "Bad").
   - Frame feedback constructively: "X% of users find the [Feature] difficult to navigate." or "Concerns regarding platform stability."

2. **Grammar & Spelling**: 
   - Labels must be professional and correctly spelled.
   - Example: "Exalent" -> "Excellent", "Custmer" -> "Customer".

3. **Summary Structure**: 
   - Every theme summary MUST be exactly two full sentences. 
   - First sentence: Define the core issue/praise.
   - Second sentence: Detail the specific impact on the user experience.

4. **Quotes**: 
   - Use full verbatim sentences for quotes to ensure context.

5. **Visuals**: 
   - Use a light-green palette for charts.
   - Include a color-to-theme legend below the chart.
